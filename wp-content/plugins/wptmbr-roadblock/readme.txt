== Description ==

This plugin uses a cookie to display a subscribe form roadblock once users have clicked through 3 pages on website. If user closes out modal without filling out form, it sets cookie that doesn not display modal for 60 days. In order for this plugin to work, it is necessary to have the Gravity Forms Plugin installed & active.
